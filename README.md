# 🚀 Harshit Bhardwaj – Developer Portfolio

A modern, high-performance personal portfolio built to showcase my journey as a **Full Stack Developer**, featuring immersive animations, interactive UI, and a curated collection of projects and technical expertise.

**🌐 Live Demo:** https://harshit-fullstack.vercel.app

---

## ✨ Features

* ⚡ Modern and responsive user interface
* 🎨 Smooth animations and transitions
* 🌌 Interactive visual effects and particle backgrounds
* 📱 Mobile-first, fully responsive design
* 🧩 Modular React component architecture
* 🚀 Fast build powered by Vite
* 💼 Dedicated sections for projects, skills, experience, and contact
* 🎯 Optimized for performance and user experience

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

### 2. Navigate to the project

```bash
cd portfolio/app
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

### 5. Build for production

```bash
npm run build
```
